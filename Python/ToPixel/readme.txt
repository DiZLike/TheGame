Полное описание параметров конфигурации

pixel_size (размер пикселя)
Тип: целое число (int)
Диапазон: 2-32
Значение по умолчанию: 8
Что делает: Определяет размер одного "квадратного" пикселя в итоговом изображении. Чем больше число, тем крупнее пиксели и сильнее эффект пиксель-арта.
Эффект:
4 — мелкие пиксели, больше деталей
8 — классический пиксель-арт (рекомендуется)
16 — очень крупные пиксели, сильное упрощение
32 — экстремально крупные пиксели (почти мозаика)

k_colors (количество цветов для K-means)
Тип: целое число (int)
Диапазон: 0-256 (0 = автоматический подбор)
Значение по умолчанию: 0
Что делает: Ограничивает количество цветов в промежуточном этапе обработки (до финальной пикселизации). K-means группирует похожие цвета в кластеры.
Эффект:
0 — авто (обычно 8-24 цвета, зависит от размера фото)
4-8 — плакатный стиль, очень плоские цвета
16 — хороший баланс (рекомендуется)
32 — больше оттенков, плавные переходы
64+ — почти фотореалистично, но может быть шумно
Примечание: Работает только с method: "smart"

edge_threshold (чувствительность краёв)
Тип: целое число (int)
Диапазон: 5-50 (должно быть нечётным)
Значение по умолчанию: 11
Что делает: Определяет, насколько агрессивно выделять границы объектов. Адаптивный порог чувствителен к контрасту между соседними пикселями.
Эффект:
5-7 — много мелких краёв, высокая детализация (может быть шумно)
9-11 — хороший баланс (рекомендуется)
13-15 — только основные границы, более гладко
20+ — очень мало краёв, объекты сливаются
Примечание: Работает только с method: "smart" и use_edge_enhance: true

saturation_boost (увеличение насыщенности)
Тип: число с плавающей точкой (float)
Диапазон: 0.5-2.5
Значение по умолчанию: 1.2
Что делает: Умножает насыщенность (интенсивность цветов) на указанное значение. Пиксель-арт обычно ярче и насыщеннее обычных фото.
Эффект:
1.0 — оригинальная насыщенность
1.2 — чуть ярче (рекомендуется)
1.5 — очень яркие, сочные цвета
2.0 — кислотные, неестественные цвета
0.8 — приглушённые цвета
Примечание: Работает со всеми методами

contrast_boost (увеличение контраста)
Тип: число с плавающей точкой (float)
Диапазон: 0.5-2.5
Значение по умолчанию: 1.1
Что делает: Умножает контраст (разницу между светлыми и тёмными участками). Пиксель-арт любит высокий контраст.
Эффект:
1.0 — оригинальный контраст
1.1-1.2 — мягкое увеличение (рекомендуется)
1.5 — очень контрастно, тени становятся чёрными
2.0 — экстремальный контраст, почти бинарное изображение
0.8 — приглушённый контраст
Примечание: Работает со всеми методами

use_nes_palette (использовать палитру NES)
Тип: логический (bool)
Значения: true / false
Значение по умолчанию: false
Что делает: Ограничивает итоговое изображение 54 цветами из оригинальной палитры NES. Принудительно заменяет все цвета на ближайшие из палитры.
Эффект:
true — аутентичный ретро-стиль NES (8-бит)
false — все цвета сохраняются (без ограничений)
Примечание: Если указан custom_palette, этот параметр игнорируется. Палитра применяется после всех остальных операций.

use_edge_enhance (усиление краёв)
Тип: логический (bool)
Значения: true / false
Значение по умолчанию: true
Что делает: Накладывает выделенные края поверх упрощённого изображения. Сохраняет чёткость границ объектов, которые иначе могли бы размыться.
Эффект:
true — чёткие границы объектов, лучше читаются формы
false — более плавные переходы, может быть "мыльнее"
Примечание: Работает только с method: "smart"

smoothing_strength (сила сглаживания)
Тип: целое число (int)
Диапазон: 0-5
Значение по умолчанию: 1
Что делает: Применяет медианный фильтр для удаления шума ДО основной обработки. Каждое значение = один проход фильтра.
Эффект:
0 — без сглаживания (может быть шумно)
1 — лёгкое удаление шума (рекомендуется)
2 — среднее сглаживание
3-5 — сильное сглаживание, теряются мелкие детали
Примечание: Полезно для шумных фото или сжатых JPEG

method (метод обработки)
Тип: строка (string)
Значения: "smart" / "edge_preserving" / "simple"
Значение по умолчанию: "smart"
Что делает: Выбирает алгоритм преобразования
